# Spam Email Detection

A simple machine learning project that classifies text messages as **Spam** or **Ham (Not Spam)** using TF-IDF and Logistic Regression.

## Features
- Text preprocessing (tokenization, vectorization)
- Model training and evaluation
- Achieves ~95% accuracy on test data
- Easy to extend to neural networks or decision trees

```bash
git clone https://github.com/<yourusername>/spam-email-detection.git
